# Телеграм-Бот для проверки позиций товара

Наш Телеграм-Бот предназначен для проверки позиций товара в поисковой выдачи на Wildberries.
Отчеты по позициям автоматически кластеризуются нашим сервисом.

## Установка

Для установки бота перейдите по ссылке - https://t.me/neuromarket_tgbot или найдите через внутренний поиск телеграма по названию @neruomarket_tgbot

## Работа с ботом

Отправьте боту сообщение с артикулом товара для проверки.

<Callout type="warning" emoji="⚠️">
Отправить необходимо только сам артикул, а не ссылку на товар
</Callout>

Бот в ответном сообщение предодставит отчет по количеству найденных кластеров. Снизу сообщения будет кнопка `Смотреть отчет`, с помощью которой можно раскрыть полноценную статистику по позициям данного артикула.
