# Установка и Обновление Расширения

## Установка

Расширение можно установить на любой браузер семейства `Chromium`. Например: Google Chrome, Яндекс.Браузер, Opera и др.

Для Установки перейдите в [Интернет-магазин Chrome](https://chromewebstore.google.com/detail/%D0%BD%D0%B5%D0%B9%D1%80%D0%BE%D0%BC%D0%B0%D1%80%D0%BA%D0%B5%D1%82-%D1%80%D0%B5%D0%BA%D0%BB%D0%B0%D0%BC%D0%B0-%D0%BD%D0%B0-wi/blkidjnfpdkelkkelhlongdbiiabfojn?hl=ru) и нажмите на кнопку `Установить`

![Установка расширения Нейромаркет](/chromeInstallPage.png)

После установки перейдите в [Новый Рекламный кабинет Wildberries](https://cmp-new.wildberries.ru) и откройте любую **Автоматическую Кампанию**. Наше расширение сразу подгрузится и отобразится внутри интерфейса WB..

## Обновление расширения

Для обновления версии расширения не требуется каких-либо действий с вашей стороны.
Обновление происходит автоматически и моментально.
