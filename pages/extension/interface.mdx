## Главная страница рекламного кабинета

![](/interface/5.jpg)

На главной странице рекламного кабинета расширение добавляет следующие дополнительные элементы интерфейса:

<img src="/interface/1.png" style={{display: "inline-block", borderRadius: 12}}/> — Данный бэйдж символизирует о том, что расширение активно

<img src="/interface/2.png" style={{borderRadius: 17}}/> Этот блок позволяет создать и сохранить API Токен для дальнейшего вывода дополнительной информации по рекламным кампаниям, используя Официальный API Wildberries.

Кнопка <ExtensionButton text={"Создать Автоматически"}/> — API токен будет создан автоматически. Будет выставлен режим "Только чтение" с правами доступа к разделам "Аналитика" и "Продвижение". Создать API Токен может только владелец кабинета поставщика. Если Вы не владелец, запросите API токен и введите его вручную.

Кнопка <ExtensionButton text={"Ввести вручную"}/> — Выведет модальное окно для ввода API Токена в ручном режиме.

<img src="/interface/3.png" style={{borderRadius: 9, marginRight: 10, marginTop: 10, float: "left"}}/> В данном модальном окне необходимо ввести API Токен. Токен предварительно будет проверен расширением на наличие необходимых прав доступа к разделам "Аналитика" и "Продвижение". Так же расширение проверит был ли установлен режим "Только чтение" при создании токена. Если все условия выполнены, то кнопка "Сохранить" станет активной.
При нажатии на кнопку "Сохранить" будет выполнен процесс проверки валидности токена и принадлежности токена к текущему поставщику. Если какие-либо условия будут не выполнены, то Вы увидите всплывающее окно с ошибкой. В случае, если все проверки были пройдены, то API Токен будет сохранен внутри Вашего браузера и будет в дальнейшем использован для вывода дополнительной информации о Рекламных Кампаниях.

<img src="/interface/4.png" style={{borderRadius: 12, marginRight: 10, marginTop: 6, float: "left"}}/> Данный бэйдж символизирует о том, что расширение активно и было подключено по API Токену, раннее сохраненному в Вашем браузере для данного поставщика. При нажатии на данный Бэйдж появится кнопка <ExtensionButton text={"Удалить API Токен"}/> Кнопка необходима в случаях, когда API токен был просрочен, либо был удален владельцем и более не активен. Удалите такой токен с помощью данной кнопки и пройдите процедуру сохранения нового токена.
